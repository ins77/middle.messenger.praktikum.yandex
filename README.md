## Мессенджер

---

### Описание

Реализация чата на курсе Миддл-фронтенд разработичик от Яндекс.практикум

### Установка
- ```npm run dev```   - запуск версии для разработчика
- ```npm run start``` - сборка билда проекта и старт сервера
- ```npm run build``` - сборка билда проекта
